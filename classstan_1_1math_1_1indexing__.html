Detailed Description

template<typename T_mat, typename T_row_index, typename T_col_index>
class stan::math::indexing_< T_mat, T_row_index, T_col_index >

Represents indexing of a matrix with two matrices of indices.

indexing(mat, col_index, row_index)[i,j] = mat[col_index[i,j], row_index[i,j]]

Template Parameters
T_matexpression type of the matrix to index
T_row_indexexpression type of the row index
T_col_indexexpression type of the column index
